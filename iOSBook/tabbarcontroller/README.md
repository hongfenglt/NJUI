# tabBarController

- UITabBarController添加控制器的方式有2种
    - 添加单个子控制器

```objc
- (void)addChildViewController:(UIViewController *)childController;
```

- 设置子控制器数组

```objc
@property(nonatomic,copy) NSArray *viewControllers;
```
